# HollowDex (Migration init)
## Description
This is an unofficial discord bot for all hollow knight © lovers

All data is saved globally. this means wherever you go you can show off your collection

*to be approved by team cherry*
## Functions
You start off with 3 crawlids

The enemies will randomly spawn in the desired channel. You can then try to engage a **fight** to capture the enemy.
If you fail at capturing the enemy, it runs away from you but stays capturable to others.

You can also **fight** someone else. if they accept, a fight will automatically take place and you will win or lose xp depending on your win or loss

You can also do **trades**. The concept is simple. You exchange enemies with another person to grow your collection even further!
## commands
> /help

Displays this markdown file

### /fight commands

> /fight {User} [Amount: default: 3] [duplicates: default: True]

Engages a fight of [Amount] enemies against {user}. [duplicates] defines if you can use the same type of enemy twice or not in a fight

> /fight add {enemy}

adds {enemy} to the battlefield
